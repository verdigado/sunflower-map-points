Displays interactive Leaflet-based maps with user-submitted markers and a counter block.

== Description ==

This plugin lets you:

- Add interactive maps using Leaflet.js
- Select map position and zoom via Gutenberg editor
- Store custom markers as Custom Post Types (\"custompoi\")
- Display a counter block showing number of markers submitted
- Export all POIs as CSV
- Fully compatible with Gutenberg

== Installation ==

1. Upload to `/wp-content/plugins/`
2. Activate via Plugins menu
3. Add \"Leaflet Formular Map\" block in Gutenberg editor
4. Add \"Leaflet Hint Counter\" block in Gutenberg editor
